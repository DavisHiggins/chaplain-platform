# Chaplain Platform

A fraternity leadership website for Phi Delta Theta NC Epsilon, built to support the Chaplain role through structured planning, Bible study organization, and brotherhood development.

## Overview

Chaplain Platform helps organize spiritual leadership responsibilities through a weekly Bible study calendar, semester scripture plan, and structured resources for accountability, growth, and chapter engagement.

## Features

- Weekly Bible study calendar
- 16-week scripture plan
- Leadership-focused content structure
- Clean fraternity-branded interface
- Responsive design
- Built for Phi Delta Theta NC Epsilon

## Tech Stack

- React
- Vite
- JavaScript
- Tailwind CSS
- Vercel
